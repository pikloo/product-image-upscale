== Description ==
Plugin Wordpress/WooCommerce permettant de redimensionner les photos des fiches produits WooCommerce

Ce Plugin requiert WooCommerce et pour son bon fonctionnement une clé API Claid AI :
https://claid.ai/account/overview

== Fonctionnalités ==

* Page Menu option :
    ** Sous menu #Paramètres généraux: Un formulaire permet de renseigner la clé API Claid AI et diverses options dont les valeurs seront envoyés ultérieurement par l'API Claid AI
    ** Sous menu #Redimensionner toutes les images: Permet de redimensionner les images des produits WooCommerce en masse et délivre un log et un résultat des fichiers traités

* Page Media List : Un bouton dans la colonne 'Product Image Upscale' permet de redimensionner unitairement les images attachés à des produits WooCommerce
